# Heart Disease Risk Predictor 🫀

A desktop GUI application for cardiovascular risk assessment, built with Python and Tkinter.  
Combines a **rule-based clinical scoring engine** (Framingham / ACC-AHA guidelines) with an optional **trained Random Forest classifier** (UCI Heart Disease dataset).

## Features

- 18 clinical parameters (demographics, vitals, labs, ECG findings, lifestyle)
- Real-time sliders, toggle buttons, and dropdowns
- Risk meter with Low / Moderate / High / Very High classification
- Top contributing risk factors with impact labels
- Plain-English summary of findings
- Optional ML model (Random Forest) — falls back to rule engine if no model is present

## Installation

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/heart-disease-predictor.git
cd heart-disease-predictor

# 2. Create a virtual environment (recommended)
python -m venv venv
source venv/bin/activate        # Linux / macOS
venv\Scripts\activate           # Windows

# 3. Install dependencies
pip install -r requirements.txt

# 4. Run the app
python app.py
```

> Tkinter is included in the Python standard library.  
> On Linux you may need: `sudo apt-get install python3-tk`

---

## Training the ML Model (optional)

Download the [UCI Heart Disease dataset](https://www.kaggle.com/datasets/redwankarimsony/heart-disease-uci) and save it as `data/heart.csv`.

```bash
python src/train_model.py --data data/heart.csv --model models/model.pkl
```

The app auto-detects `models/model.pkl` on startup and switches from the rule engine to ML predictions.

## Input Parameters

| Category | Parameters |
|---|---|
| Demographics | Age, Sex, BMI |
| Vitals & Labs | Resting BP, Cholesterol, Resting HR, Max HR, Fasting Blood Sugar, ST Depression |
| Clinical Findings | Chest Pain Type, Resting ECG, ST Slope, Exercise Angina, Major Vessels, Thalassemia |
| Lifestyle & History | Smoking Status, Diabetes, Family History of CVD |
